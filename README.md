# Installation

## Clone Repository

```bash
git clone https://github.com/AbhayUpadhyay1112/Team-Task-Manager.git
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# Backend Setup

```bash
cd backend
npm install
node server.js
```

---

# Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000
DATABASE_URL=your_postgresql_url
JWT_SECRET=your_secret_key
```
